# DingTalk Messager
Chrome Extension
Unoffical Messager for [im.dingtalk](https://im.dingtalk.com/) , show all messages as desktop notifications

# FEATURES
- Message Avatar
- Messages from same source merged
- Quick jump back to im.dingtalk
